PROJECT ON CONSOLE SCREEN

First of all, open VISUAL STUDIO CODE and right click on the empty bar on the left side of your computer screen. Then click on "NEW FILE" option and give it a name and then START CODING!!

CONSOLE SCREEN IS USED TO STORE SOME IMPORTANT DATA OR CALCULATION WHILE MAKING A WEBSITE. WE CAN OPEN CONSOLE SCREEN BY RIGHT CLICKING ANYWHERE ON THE WEBPAGE AND SELECT THE OPTION "INSPECT" AND THEN CLICK ON THE OPTION CALLED CONSOLE AND THERE IT IS YOUR CONSOLE SCREEN!

WE CAN PRINT ANYTHING ON THE CONSOLE SCREEN BY A SMALL JAVA CODE "console.log"
